# PyStudy - 인터랙티브 Python 학습 플랫폼

초보자를 위한 웹 기반 대화형 파이썬 학습 사이트입니다.
별도의 설치 없이 브라우저에서 직접 파이썬 코드를 작성하고 실행하며 학습할 수 있습니다.

## 🚀 주요 기능

- **웹 기반 코드 실행**: Pyodide를 통합하여 서버 없이 브라우저 내에서 즉시 파이썬 코드를 실행할 수 있습니다.
- **체계적인 커리큘럼**: 기초 문법부터 제어문, 함수까지 단계별로 구성된 학습 챕터를 제공합니다.
- **프리미엄 대시보드 UI**:
  - 눈이 편안한 다크 모드 기반 디자인
  - 모바일에서도 학습 가능한 반응형 사이드바
  - 실시간 실행 결과 콘솔
- **직관적인 에디터**: 코드 하이라이팅과 자동 들여쓰기를 지원하는 에디터 경험을 제공합니다.

## 🛠 기술 스택

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Engine**: [Pyodide](https://pyodide.org/) (WebAssembly 기반 Python 런타임)
- **Fonts**: Inter, Noto Sans KR, Fira Code (코딩 전용 폰트)

## 📂 프로젝트 구조

```
w:/web/python_study/
├── css/
│   └── style.css       # 다크 모드 및 반응형 스타일 시트
├── js/
│   ├── chapters.js     # 학습 커리큘럼 및 예제 코드 데이터
│   └── script.js       # Pyodide 연동 및 사이트 로직
├── index.html          # 메인 페이지 (대시보드 구조)
└── readme.md           # 프로젝트 설명서
```

## 💻 실행 방법

1. `w:/web/python_study/` 폴더로 이동합니다.
2. `index.html` 파일을 웹 브라우저(Chrome, Edge 등)로 엽니다.
3. **인터넷 연결이 필요합니다.** (Pyodide 엔진을 CDN에서 로드하기 위함)
4. 좌측 메뉴에서 챕터를 선택하고, 우측 에디터에서 코드를 실행하며 학습합니다.
